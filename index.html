The Process

Whilst searching for jQuery scripts, I came across Hook.js and fell in love with its website. I tried to recreate the flat, simplistic style, and employ the same subtle negative margin on the screenshot. I added my own playfulness to the website by using CSS3 transformations.

Another objective I had whilst creating this site was to use SASS for the first time. Using Coda to write the HTML and CSS, and using Codekit to compile the .scss files into the main.css file made organisation of my CSS a breeze. You can view the original .scss files here.
